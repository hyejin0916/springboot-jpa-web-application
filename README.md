# springboot-jpa-web-application

## Repository Status
This repository represents the baseline web application implementation.  
Further performance optimization and API refactoring are available in the springboot-jpa-web-application-optimized repository.

## Features
- Spring Boot 기반 웹 애플리케이션
- JPA를 활용한 엔티티/DB 연동
- CRUD 기능 구현 (상품, 회원 등)
- Thymeleaf 기반 뷰 렌더링

## Notes
- This repository is fixed as the baseline version.
- All improvements, performance optimization, and API-based refactoring will be in the springboot-jpa-web-application-optimized repository.
